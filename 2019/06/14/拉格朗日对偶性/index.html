拉格朗日对偶性

发表于 2019-06-14 | 分类于 机器学习 | 阅读次数:

在约束最优化问题中，常常利用拉格朗日对偶性将原始问题转换为对偶问题，通过解对偶问题而得到原始问题的解。

该方法应用在许多统计学习方法中，例如，最大熵模型与支持向量机。

9.1 原始问题

假设，，是定义在上的连续可微函数。考虑约束最优化问题：

L(x,\alpha,\beta) = f(x)+\sum{i=1}^k \alpha_i c_i(x)+\sum{j=1}^l \beta_j h_j(x) \ (9.4)

\thetap(x) = \max{\alpha,\beta:\alpha_i \ge 0} L(x,\alpha,\beta) \ (9.5)

\thetap(x) = \max{\alpha,\beta:\alphai\ge0} \left[f(x)+\sum{i=1}^k\alphaic_i(x)+\sum{j=1}^l\beta_jh_j(x)\right] = +\infty \ (9.6)

\theta_p(x) = \left{\begin{aligned} f(x) , \ & x satisfy the constraints \ +\infty, \ & others \end{aligned}\right. \ (9.7)

\minx \theta_p(x) = \min_x \max{\alpha,\beta:\alpha_i\ge0} L(x,\alpha,\beta) \ (9.8)

p^* = \min_x \theta_p(x) \ (9.9)

\theta_D(\alpha,\beta) = \min_x L(x, \alpha, \beta) \ (9.10)

\max{\alpha,\beta:\alpha_i\ge0} \theta_D(\alpha,\beta) = \max{\alpha,\beta:\alpha_i\ge0} \min_x L(x,\alpha,\beta) \ (9.11)

d^* = \max_{\alpha,\beta:\alpha_i \ge0} \theta_D(\alpha, \beta) \ (9.14)

d^ = \max{\alpha,\beta:\alpha_i\ge0} \min_x L(x,\alpha,\beta) \le \min_x \max{\alpha,\beta:\alpha_i\ge0} L(x,\alpha,\beta) = p^ \ (9.15)

\thetaD(\alpha,\beta) = \min_x L(x,\alpha,\beta) \le L(x,\alpha,\beta) \le \max{\alpha,\beta:\alpha_i \ge 0} L(x,\alpha,\beta) = \theta_p(x) \ (9.16)

\theta_D(\alpha,\beta) \le \theta_p(x) \ (9.17)

\max_{\alpha,\beta:\alpha_i\ge0}\theta_D(\alpha,\beta) \le \min_x \theta_p(x) \ (9.18)

d^ = \max{\alpha,\beta:\alpha_i \ge 0} \min_x L(x,\alpha,\beta) \le \min_x \max{\alpha,\beta:\alpha_i\ge0}L(x,\alpha,\beta) = p^ \ (9.19)

p^ = d^ = L(x^,\alpha^,\beta^*) \ (9.20)

\nabla_x L(x^,\alpha^,\beta^*) = 0 \ (9.21)

\alpha_i^c_i(x^) = 0 , i=1,2,…,k \ (9.22)

c_i(x^*) \le 0 , i=1,2,…,k \ (9.23)

\alpha_i^* \ge 0 , i=1,2,…,k \ (9.24)

h_j(x^*) = 0 , j=1,2,…,l \ (9.25)

$$

特别指出，式(9.24)称为KKT的对偶互补条件。由此条件可知：若，则。
